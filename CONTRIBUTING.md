# المساهمة

1. عدّل إضافة واحدة قدر الإمكان في كل Pull Request.
2. ارفع `extVersionCode` في ملف `build.gradle` الخاص بالإضافة.
3. شغّل `.\gradlew.bat -p src assembleDebug` على Windows أو `./gradlew -p src assembleDebug` على Linux/macOS.
4. اختبر البحث، التفاصيل، قائمة الحلقات، واستخراج الفيديو على أحدث إصدار Aniyomi.
5. لا تضف مفاتيح API خاصة، cookies، حسابات، آليات تجاوز DRM، أو ملفات توقيع.

أي مصدر جديد يجب أن يكون عربي الواجهة أو المحتوى، وأن يحمل `lang = "ar"`، وأن يتضمن أيقونة وشفرة قابلة للبناء.
