# حالة النطاقات

فحص HTTP أساسي بتاريخ 6 يوليو 2026. هذا الفحص لا يشغّل فيديو ولا يتجاوز حماية آلية، لذلك يصف الوصول إلى النطاق فقط.

| المصدر | نتيجة الفحص | ملاحظة |
|---|---|---|
| Anime4Up | يصل بعد إعادة توجيه | النطاق النهائي متغير؛ يحتاج اختبارًا من داخل Aniyomi |
| Anime Blkom | HTTP 403 | النطاقات الثلاثة المضمّنة تستجيب لكن خلف حماية آلية |
| Animeiat | API متوقف | `animeiat.tv` أصبح صفحة parking وواجهة `api.animeiat.co` لا تستجيب؛ Legacy |
| AnimeLek | الموقع انتقل بتصميم مختلف | `animelek.ma` يستجيب لكن محدداته لا تطابق الإضافة الحالية؛ Legacy حتى إعادة كتابة المحلل |
| Animerco | HTTP 200 | حُدّث إلى `eta.animerco.org` وتأكد وجود محددات القائمة الحالية |
| ArabAnime | HTTP 403 | الموقع موجود خلف حماية آلية |
| Arab Seed | الموقع انتقل بتصميم مختلف | توجد نطاقات حية جديدة، لكن بنية الصفحة لا تطابق الإضافة الحالية؛ Legacy حتى إعادة كتابة المحلل |
| Asia2TV | HTTP 200 | النطاق استجاب |
| Cimaleek | HTTP 200 | حُدّث إلى `r101.cimalek.buzz`؛ يلزم اختبار التشغيل من التطبيق |

| FASELHD | HTTP 403 | حُدّث إلى `fasel-hd.cam` لكن الحماية منعت اختبار المحددات |
| MY CIMA | النطاق القديم مغلق | `wecima.press` يستجيب بتصميم مختلف لا تطابقه محددات الإضافة الحالية؛ Legacy حتى إعادة كتابة المحلل |
| Okanime | HTTP 200 | حُدّث إلى `ww3.okanime.xyz`؛ يلزم اختبار التشغيل من التطبيق |
| WIT ANIME | HTTP 200 | حُدّث إلى `witanime.xyz`؛ يلزم اختبار التشغيل من التطبيق |

نجاح البناء منفصل عن نجاح المصدر وقت التشغيل. الإضافات غير المتاحة أُبقيت كقاعدة صيانة تاريخية بدل حذف شفرتها.
