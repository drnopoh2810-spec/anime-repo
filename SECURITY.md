# Security

لا تثبّت APK إلا إذا بنيته بنفسك أو نزّلته من فرع `repo` الخاص بهذا المستودع بعد مراجعة Workflow. لا تُرسل تقارير أمنية تتضمن كلمات مرور أو مفاتيح توقيع في Issue عامة.

أبلغ عن السلوك غير المتوقع أو الطلبات الشبكية غير المرتبطة بالمصدر عبر Security Advisory خاص في GitHub. المشروع لا يخزّن بيانات دخول أو مفاتيح API أو cookies ثابتة.
